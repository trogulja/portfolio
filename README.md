# Personal folio site
here be some code for personal site - still under construction

## Build Setup
VueJS + Vuetify static website.


```bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn serve

# run linter
yarn lint

# build for production
yarn build
```
